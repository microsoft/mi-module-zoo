A relational transformer encoder layer. That supports both discrete/sparse edge types and dense (all-to-all) relations, different ReZero modes, and different normalization modes.

Parameters

  • d_model â the dimensionality of the inputs/ouputs of the transformer layer.

  • key_query_dimension â the dimensionality of key/queries in the multihead attention.

  • value_dimension â the dimensionality of the multihead attention values,

  • num_heads â the number of attention heads,

  • num_edge_types â the number of discrete edge types. If 0, no discrete edge types are to be used.

  • add_reverse_edges â if num_edge_types>0 should reverse edge types be introduced?

  • dim_feedforward â the dimensionality of the feedforward hidden layer in the transformer layer.

  • dropout_rate â the dropout rate in \([0, 1)\),

  • activation â the activation function to be used in the feedforward layer. Defaults to ReLU.

  • use_edge_value_biases â should the discrete edges (relations) use value biases?

  • edge_attention_bias_is_scalar â should edge_attention_biases be a scalar or of size key_query_dimension?

  • rezero_mode â

    Three different modes are supported

    • "off": No ReZero use.

    • "scalar": Sublayers (attention / fully connected) are scaled by a single scalar, i.e., alpha is a scalar in the following:

      x' = x + alpha * SelfAtt(x)
x'' = x' + alpha * Boom(x')
return x''

      See https://arxiv.org/pdf/2003.04887.pdf.

    • "vector": Sublayers (attention / fully connected) are scaled by one value per dim, i.e., alpha is a vector in the following:

      x' = x + alpha * SelfAtt(x)
x'' = x' + alpha * Boom(x')
return x''

      See https://arxiv.org/pdf/2103.17239.pdf.

  • normalisation_mode â

    Three different modes are supported:

    • "off": use no layer norm at all. Likely to diverge without using ReZero as well.

    • "prenorm": Normalise values before each sublayer (attention / fully connected):

      x' = x + SelfAtt(LN(x))
x'' = x' + Boom(LN(x'))
return x''

    • "postnorm": Normalise values after each sublayer:

      x' = LN(x + SelfAtt(x))
x'' = LN(x' + Boom(x))
return x''
